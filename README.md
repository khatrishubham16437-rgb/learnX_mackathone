# üöÄ LearnX | Neural Adaptive Learning Academy

**LearnX** is a futuristic, AI-driven learning platform designed to bridge the gap between static tutorials and interactive mastery. It combines a high-end 3D visual interface with a real-time **Neural Practice Engine** powered by **Llama 3.2**.



## üåü Key Features

* **Neural Practice (AI Quiz):** Generates real-time, unique C++ MCQs using **Ollama & Llama 3.2**. No two questions are ever the same.
* **Adaptive XP System:** Gain **+20 XP** for correct answers and face a **-10 XP penalty** for mistakes to ensure focused learning.
* **Streak Tracking:** Maintain a winning streak to prove your consistency. One wrong move resets your streak to zero.
* **The Architects (Mentors):** A dedicated dashboard to explore and connect with elite industry mentors.
* **Global Ranks:** A competitive leaderboard showcasing top-performing students based on their XP and streaks.
* **Immersive 3D UI:** Built with **Three.js** and **GSAP**, featuring a starfield hyperspace intro and interactive 3D badges.

## üöÄ Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/shubhamkhatri16437/LearnX.git](https://github.com/shubhamkhatri16437/LearnX.git)
    cd LearnX
    ```

2.  **Setup the AI Engine:**
    * Install [Ollama](https://ollama.com/)
    * Pull the Llama 3.2 model:
        ```bash
        ollama pull llama3.2
        ```

3.  **Install Backend Dependencies:**
    ```bash
    npm install
    ```

4.  **Start the Server:**
    ```bash
    node server.js
    ```

5.  **Run the App:**
    Open `login.html` in your browser to begin the neural experience.
